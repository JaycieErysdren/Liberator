Liberator is an Electron app utilizing KaitaiStruct, three.js, jsTree and jQuery. It can load, view, convert and extract data from various obscure binary file formats.

Files from the following engines are supported:

  • BRender
  • Descent
  • idTech 1
  • Microsoft 3D Movie Maker
  • SlaveDriver
  • Tank Engine
